# Expense Tracker

A simple JavaScript Expense Tracker that allows users to manage daily spending.

## Features

Add expenses  
Delete expenses  
Filter expenses by category  
Calculate total spending  
Store data using LocalStorage  
Unit testing using Jest  

## Technologies Used

HTML  
Tailwind CSS  
JavaScript  
LocalStorage  
Jest  

## Setup

Clone the repository

git clone <repo-link>

Install dependencies

npm install

Run tests

npm test

Open index.html in your browser.
